ADB_BOX工具
功能：
1. Android设备动态拉取、截屏、录屏
2. 触摸屏幕
3. 虚拟遥控器
4. 远程输入
5. Logcat抓取，root
6. 设备属性、信息、当前运行APP信息实时获取
7. 应用定位、应用包名、版本号等信息实时获取
8. 设备文件管理（查看、上传、下载）
